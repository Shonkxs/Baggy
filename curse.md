# Baggy - Modern Category Bags for WoW Retail

Tired of digging through chaotic bags every time you press **B**?  
**Baggy** replaces the default bag UI with a clean, modern window that helps you find items faster and keep your inventory under control.

## Why players use Baggy

- **Smart categories** for quick access:
  - Consumables
  - Armor (with slot-based subtabs)
  - Materials
  - Gems
  - Enchantments
  - Mounts
  - Misc
- **Instant search** in the active tab
- **Bank mode** to view inventory + bank together while at the bank
- **Gold display** with WoW coin icons
- **Tracked currencies** in the header (add/remove with one click)
- **Crafting quality badges** on supported reagent icons
- **English UI in every game locale** (classification is API-based, not tooltip text parsing)

## Built for daily gameplay

Baggy is designed to be fast, readable, and easy to use in real gameplay:
- Open bags with normal keybinds/buttons
- Keep your preferred frame size/position
- Keep your active tab and layout settings between sessions

## Slash Commands

- `/baggy` - Toggle Baggy window
- `/baggy help` - Show command help
- `/baggy reset` - Reset layout and settings
- `/baggy lock` - Toggle frame lock
- `/baggy scale <0.75-1.50>` - Set frame scale
